## [0.1.0] - 2026-05-06

### Added
- Парсер Kwork.ru через официальный API с постраничным обходом
- Rule-based scoring: фильтрация по ключевым словам, бюджету, категориям и рейтингу заказчика
- AI-анализ релевантности через OpenRouter (опционально, любая модель)
- Генерация черновиков откликов через отдельную AI-модель
- Telegram Bot с inline-кнопками: Интересно / Мимо / Скрыть похожие / Отклик
- Команда `/health` — состояние процесса и статистика
- SQLite хранилище с миграциями и отслеживанием feedback
- Фильтр "Скрыть похожие" на основе TF-IDF сравнения описаний
- Docker & Docker Compose поддержка
- Режим `--once` для разового запуска / тестирования
- `KWORK_DRY_RUN=true` для проверки фильтров без отправки в Telegram
- Конфигурация через `.env` без изменения кода
- 41 unittest, покрывающий scoring, parsing, storage, telegram feedback

### Known Issues
- Kwork имеет ограничения на автоматизацию — рекомендуется интервал 45–60 сек между запросами
